# Repository Policy

This repository is public for research documentation,
development tracking, and reproducibility.

The following must never be committed:

- Unreal Engine source code or binaries
- Cesium ion access tokens
- Google or other map-provider API keys
- Downloaded 3D Tiles caches
- Private credentials
- Generated large-scale datasets
- Restricted third-party assets

Unreal Engine, Cesium for Unreal, Colosseum, and map data
providers remain subject to their own licenses and terms.

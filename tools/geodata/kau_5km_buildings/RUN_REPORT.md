## 높이 품질

| 높이 결정 방식 | 건물 수 | 비율 |
|---|---:|---:|
| 브이월드 `height` 실측/등록값 | 16,695 | 30.913% |
| `grnd_flr × 3.2 m` | 11,344 | 21.005% |
| 높이·층수 없음, 9.6 m 임시값 | 25,968 | 48.083% |

최종 높이의 중앙값은 9.6 m, 95백분위는 19.2 m, 최댓값은 150.0 m다.
건물 외곽은 공식 데이터 전량이지만, 높이의 48.083%는 공식 높이가 아닌
임시값이므로 이후 3D 실험에서 반드시 `height_source`로 구분해야 한다.

`N37E126.hgt.gz`를 결합했다. EGM96 평균해수면고도는 시스템
`egm96_15.gtx` 모델로 WGS84 타원체고도로 변환했다.

`validate_lod1_tiles.py`는 316개 GLB의 헤더·buffer·index와 ECEF 변환
948개 표본을 검사했으며 모두 통과했다. Unreal Editor 빌드도 성공했고,
`HighAltitudeCity` 런타임 시험에서 `KAU_Buildings_5km` Actor가 자동
생성되어 `tileset.json`과 316개 GLB가 HTTP 200으로 로드되는 것을 확인했다.

지면고도는 30 m급 SRTM의 건물 대표점 주변 5×5 셀 25백분위로 추정한
MVP 값이다. 실제 Cesium World Terrain과의 수직 차이는 다음 시각 검증
단계에서 측정해야 한다.
